readerについて
-------
点は空白文字で区切られた数字列で与えること.  
奇数個与えた場合、末尾は切り捨てる.

    x-1 y-1
    x-2 y-2
    x-3 y-3
    x-4 y-4

printerについて
------
点毎にx,y成分を空白区切りで一行ずつ出力する.  
区切り文字を変えたい場合や、行末に文字列を追加したい場合は、
`pset_raw_delimiter`や`pset_col_delimiter`を変更する.
